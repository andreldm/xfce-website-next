--- title: "About" ---

Xfce is a lightweight desktop environment for UNIX-like operating systems. It aims to be fast and low on system resources, while still being visually appealing and user friendly.

Xfce embodies the traditional UNIX philosophy of modularity and re-usability. It consists of a number of components that provide the full functionality one can expect of a modern desktop environment. They are packaged separately and you can pick among the available packages to create the optimal personal working environment.

Another priority of Xfce is adherence to standards, specifically those defined at freedesktop.org.

Xfce can be installed on several UNIX platforms. It is known to compile on Linux, NetBSD, FreeBSD, OpenBSD, Solaris, Cygwin and MacOS X, on x86, PPC, Sparc, Alpha...

Features

Xfce contains a number of core components for the minimum tasks you'd expect from a desktop environment:

Window Manager

Manages the placement of application windows on the screen, provides window decorations and manages workspaces or virtual desktops. Learn More →

Panel

The panel allows you to switch between opened windows, launch programs and show plugins like a menu, a clock, a desktop switcher and more. Learn More →

Desktop Manager

Sets the background image and provides a root window menu, desktop icons or minimized icons and a windows list. Learn More →

Session Manager

Controls all aspects of the startup and the shutdown of the user session and power management. Learn More →

Settings Manager

It allows you to configure every detail of your desktop environment, like keyboard shortcuts, appearance, display settings and much more. Learn More →

Thunar File Manager

Thunar is a modern file manager designed from the ground up to be fast and easy-to-use and has unique utilities like the bulk renamer. Learn More →

Application Finder

Allows you to quickly search, launch and find information about applications installed on your system. Learn More →

Power Manager

Manages the power sources of the computer and connected devices. Learn More →

Xfce Libraries

Widgets library for the Xfce desktop environment. Learn More →

Xfconf

Simple client-server configuration storage and query system. Learn More →
