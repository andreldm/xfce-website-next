--- title: "Projects" --- The Xfce project contains several separated projects for each part of the desktop. Some projects have their own project pages to provide additional information.

Core Components

Window Manager

Manages the placement of application windows on the screen, provides window decorations and manages workspaces or virtual desktops. Learn More →

Panel

The panel is a bar which allows you to have at all times program launchers, panel menus, a clock, a desktop switcher and more. Learn More →

Desktop Manager

This program sets the background image and provides a root window menu, desktop icons or minimized icons and a windows list. Learn More →

Session Manager

The session manager controls all aspects of the startup and the shutdown of the user session. Learn More →

Settings Manager

It allows you to configure every detail of your desktop environment. Learn More →

Thunar File Manager

Thunar is a modern file manager designed from the ground up to be fast and easy-to-use. Learn More →

Application Finder

Allows you to search, launch and find information about applications installed on your system. Learn More →

Power Manager

Manages the power sources of the computer and connected devices. Learn More →

Xfce Libraries

Widgets library for the Xfce desktop environment. Learn More →

Xfconf

Simple client-server configuration storage and query system. Learn More →
